## [1.0.0] - 2025-03-03

### Anadido
- Disenador visual con canvas drag & drop y 8 handles de redimensionado
- Tipos de elemento: Texto editable/estatico, Imagen, Video, Rectangulo, Elipse
- Panel de propiedades: posicion, tamano, tipografia, color, animaciones CSS
- Resoluciones: FHD, 4K, HD, Vertical y personalizado
- Fuente CSV con deteccion automatica de cabeceras, filtrado y preview
- Fuente API REST con peticion periodica configurable
- Fuente SSE con conexion persistente y reconexion automatica
- Fuente WebSocket con reconexion automatica
- Autenticacion: Sin auth, API Key, Bearer Token, Basic Auth
- Mapeo JSON: Ruta JSON, Filtrar array, Template con {campo}
- Sistema CORS con 4 niveles de fallback automatico
- Export HTML con CSS, animaciones y JS de fuentes de datos
- Gestion de proyectos (localStorage + ficheros .nsignproj)
- Auto-guardado cada 2 minutos
- Panel de logs con 5 niveles
- Boton Test para probar conexiones en tiempo real
- Version visible en el toolbar (v1.0.0)
- Importar proyectos directo desde toolbar
- Fix imagenes: wrapper div para compatibilidad nsign
